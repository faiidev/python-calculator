# 🧮 Python Calculator V1

## 📖 Description

Python Calculator V1 adalah mini project yang dibuat untuk mempelajari dasar-dasar Python, khususnya penggunaan variabel, input/output, dan operator aritmatika.

## ✨ Features

- Penjumlahan (+)
- Pengurangan (-)
- Perkalian (*)
- Pembagian (/)
- Sisa Bagi (%)
- Pangkat (**)
